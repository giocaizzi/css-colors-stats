# css-colors-stats

Quickly extract and count colors from CSS files.

Available color formats:
- `hex`
- `rgb`
- `rgba`
- `hsl`
- `hsla`

## Installation

Clone this repo and install it with *pip*.

```shell
git clone <this-repo>
pip install .
```

## Usage

```shell
css-colors-stats <path-to-css-file>
```
